# Meus Sites Favoritos

Página web desenvolvida com HTML5 que exibe meus sites favoritos, com suas respectivas logos e links clicáveis para acesso direto.

## Tecnologias utilizadas

- HTML5

## Como executar

1. Clone ou baixe o repositório
2. Certifique-se de que as imagens estão na mesma pasta do arquivo `index.html`
3. Abra o arquivo `index.html` em qualquer navegador
